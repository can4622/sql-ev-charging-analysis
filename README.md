⚡ Electric Vehicle (EV) Charging Station Analysis with SQL
Bu proje, hızla büyüyen elektrikli araç (EV) pazarındaki şarj istasyonu verilerini analiz etmek, kullanım trendlerini belirlemek ve altyapı ihtiyaçlarını öngörmek amacıyla hazırlanmıştır.

📊 Veri Seti Hakkında
Analiz süreci, şarj istasyonlarının operasyonel verilerini içeren şu temel metrikler üzerinden yürütülmüştür:

Station Locations: Bölgesel dağılım ve yoğunluk analizi.

Usage Patterns: Günün farklı saatlerindeki doluluk oranları.

Charging Types: Hızlı (DC) ve standart (AC) şarj ünitelerinin kullanım tercihleri.

Energy Consumption: İstasyon bazlı enerji tüketim trendleri.

🎯 Analiz Hedefleri
Bu çalışmada SQL sorguları ile şu kritik sorulara yanıt aranmıştır:

Hangi bölgelerde şarj istasyonu altyapısı talebi karşılamakta yetersiz kalıyor?

Şarj istasyonlarının en yoğun kullanıldığı zaman dilimleri nelerdir?

Mevcut şarj ağının büyüme hızı, elektrikli araç satış trendleri ile uyumlu mu?
